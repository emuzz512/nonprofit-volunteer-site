# Impact Systems Studio

A volunteer website offering systems design and process improvement support to nonprofits and service organizations.

## About

This site showcases professional volunteer services focused on:
- Process streamlining and optimization
- Cross-functional team alignment
- Strategic planning and implementation
- Systems design for sustainability

## Technology

- Pure HTML5, CSS3, and vanilla JavaScript
- No frameworks or dependencies
- Fully responsive design
- Award-winning aesthetic inspired by modern web design
- Optimized for performance and accessibility

## Deployment

This site is designed to be deployed via GitHub Pages:

1. Push to GitHub
2. Enable GitHub Pages in repository settings
3. Select main branch as source
4. Site will be live at `https://yourusername.github.io/nonprofit-volunteer-site`

## Customization

Before deploying, update:
- Email address in `index.html` (line with `mailto:`)
- LinkedIn profile URL in `index.html`
- Any personal details or messaging

## Local Development

Simply open `index.html` in a browser to view locally.
